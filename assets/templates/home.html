Welcome!

The website hosts automatically created card rotation charts and statistics for the cards from Love Live SIF All Stars, updated daily. These charts can help you figure out which girl is most likely to receive a card next based on statistics like how many cards are remaining in the current rotation and how long it's been since their last card.

Of course keep in mind that the developers are bound by no rule when it comes to adding new content, something that is decidedly evident when it comes to events. The site can at most serve you to make educated guesses.

Card and event data is retrieved from Kirara All Stars Card Database. They automatically scrape new card and event data which can mean that sometimes the data on their site, and consequently this one, can be incorrect or out of date as a result. A best effort is made and corrections will be made promptly.

Card release and event timestamps can be toggled between the original Japanese server releases and Global version dates, before acceleration ended. Check the "Use Global Dates" toggle in the settings.

If you have feedback, suggestions or error reports regarding this website you can send feedback.

Newest Updates

Previous Card Elapsed Time on Tooltips

Applicable card rotation pages now display the number of days elapsed since the previous card was released in the tooltips on hover (desktop) or on tap (mobile).

This is a nice little quality of life update to save a trip to the card history section .

Member Selection with a Keyboard

Good news for those visitors with a physical keyboard: the member selection boxes event history and banner history now accept keyboard inputs giving you a quicker way to select the girl whose history you're interested in.

Pressing the Enter-key will now open the selection box and once it is open you can start typing a name. The full name is matched against alongside some common, less common and completely made-up aliases. See if you can figure some out. Once you have a match you can press Enter-key again to select the first highlighted girl in the list.

The Timeline is Here!

At long last, the timeline is ready for public release!

The timeline features a full release history of all cards, events and banners from the game launch up until today and displays the relative release dates of all cards. You can view the timeline with both the original Japanese dates and the global dates, and swapping between the two from site settings will keep roughly equivalent position in terms of card releases.

For the best experience, using the timeline on a more powerful device with a large screen (e.g. desktop PC) is strongly recommended, as it is rather heavy especially on low performance mobile devices.

As always bug reports are welcome!

Getting started

Just navigate to any of the categories in the navigation bar on the left.

Note that you can also operate some settings by keybindings:

  • General
  • Q Switches between idolized and normal card icons.
  • W Reverses the order of lists.
  • D Switches between Japan server and global card release, event and banner dates.
  • S Cycles to the next source highlight option. Use with the Shift key to go backwards.
  • R Resets current source highlighting.
  • T Toggles the tooltip visibility.
  • Number keys Changes the active sub-page when applicable.
  • Card Stats
  • C Collapses stats table groups, displaying all members in a single table instead.
  • H Hides rows for members that have not yet received a card in the viewed category.
  • Event and Banner History
  • E Cycles to the next filtered member. Use with the Shift key to go backwards.
  • H Highlights cards of the filtered member. Overrides the default highlighting setting.
  • Enter Opens and closes the member selection box. When open you can type in a name of any girl you wish to quick select.
    • Only on Event History
    • F Displays only events with the featured member being the one currently selected.
    • Only on Banner History
    • B Cycles to the next filtered banner. Use with the Shift key to go backwards.
  • R In addition to the default, will also reset options mentioned above when viewing the event or banner history.
  • Timeline
  • E Toggles visiblity of event runtime highlighting in the timeline.
  • B Toggles visiblity of banner runtime highlighting in the timeline.
  • F Cycles between the timeline displaying all cards, only URs or only SRs.
  • R In addition to the default, will also reset options mentioned above when viewing the timeline.

Just navigate to any of the categories in the navigation available by tapping the bars icon in the top bar.

Some site settings can be modified from the settings menu available by tapping the cog icon.
