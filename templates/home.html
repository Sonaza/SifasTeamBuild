Welcome!

The website hosts automatically created card rotation charts and statistics for the cards from Love Live SIF All Stars, updated daily. These charts can help you figure out which girl is most likely to receive a card next based on statistics like how many cards are remaining in the current rotation and how long it's been since their last card.

Of course keep in mind that the developers are bound by no rule when it comes to adding new content, something that is decidedly evident when it comes to events. The site can at most serve you to make educated guesses.

Card and event data is retrieved from Kirara All Stars Card Database. They automatically scrape new card and event data which can mean that sometimes the data on their site, and consequently this one, can be incorrect or out of date as a result. A best effort is made and corrections will be made promptly.

All card release and event timestamps used on this site are according to the original Japanese server releases.

If you have feedback, suggestions or error reports regarding this website you can send feedback.

Newest Updates

Mobile Layout Improvements

Multiple mobile layout improvements were added that should help with usability on mobile and touch-only devices.

  • Navigation and settings menus got a visual upgrade.
  • Rotation and SR set columns for the groups are now side by side, and horizontally scrollable. No need to scroll way down to find Aqours and Nijigasaki anymore.
  • Tooltips are now available on mobile and touch-only devices also: if enabled in settings, when tapping on a card icon a tooltip dialog will appear at the bottom of the screen. If you wish to still use direct links to All Stars Kirara Database instead you can disable the tooltips.

New Card History Page

Now you can browse an idol's full card history in the new card history page. The history shows the time elapsed since each card was released, as well as the time between the release of each card in the same category. This can give you an idea of how regularly the idol has received new cards in the past.

Getting started

Just navigate to any of the categories in the navigation bar on the left.

Note that you can also operate some settings by keybindings:

  • General
  • Q Switches between idolized and normal card icons.
  • W Reverses the order of lists.
  • S Cycles to the next source highlight option. Use with the Shift key to go backwards.
  • R Resets above settings back to default.
  • T Toggles the tooltip visibility.
  • Number keys Changes the active sub-page when applicable.
  • Card Stats
  • C Collapses stats table groups. Members are displayed in a single table instead of by group.
  • H Hides rows for members that have not yet received a card in the viewed category.
  • Event Cards & Banners
  • E Cycles to the next filtered member. Use with the Shift key to go backwards.
  • F Displays only events with the featured member being the one currently selected.
  • H Highlights currently selected member. Overrides the default highlighting setting.
  • R Also resets these options when currently on the Event Cards page.

Just navigate to any of the categories in the navigation available by tapping the bars icon in the top bar.

Some site settings can be modified from the settings menu available by tapping the cog icon.
