Welcome!

This site hosts automatically formatted rotation charts of the cards from Love Live SIF All Stars. These charts can be useful for determining the girl that is the most likely to receive a card in the future, based on statistics such as how many cards are remaining in the current rotation and how long it has been since their last one.

Of course the developers are not bound to rigid rotations, a fact most evident in event card rotations but also frequently seen in other categories of cards. Purpose of this site is to help make educated guesses.

Card and event data is retrieved from Kirara All Stars Card Database. Their data source is automatically scraped which does not guarantee data validity at all times so sometimes data on this site can also be incorrect or out of date as a result. A best effort is made and corrections will be made promptly.

Any timestamps used on this site are according to the original Japanese server releases.

If you have feedback, suggestions or bug reports regarding this website you can send feedback.

This website may still be a work in progress so things may be subject to change.

New Card History Page

Now you can browse an idol's full card history in the new card history page. The history displays the time elapsed since the release of a card but also the time elapsed in-between its release and the previous card in the same category giving some clue as to how regularly the idol has received new cards in the past.

Getting started

Just navigate to any of the categories in the navigation bar on the left.

Note that you can also operate some settings by keybindings:

  • General
  • Q Switches between idolized and normal card icons.
  • W Reverses the order of lists.
  • S Cycles to the next source highlight option. Use with the Shift key to go backwards.
  • R Resets above settings back to default.
  • T Toggles the tooltip visibility.
  • Number keys Changes the active sub-page when applicable.
  • Card Stats
  • C Collapses stats table groups. Members are displayed in a single table instead of by group.
  • H Hides rows for members that have not yet received a card in the viewed category.
  • Event Cards & Banners
  • E Cycles to the next filtered member. Use with the Shift key to go backwards.
  • F Displays only events with the featured member being the one currently selected.
  • H Highlights currently selected member. Overrides the default highlighting setting.
  • R Also resets these options when currently on the Event Cards page.

Just navigate to any of the categories in the navigation available by tapping the bars icon in the top bar.

Some site settings can be modified from the settings menu available by tapping the cog icon.
