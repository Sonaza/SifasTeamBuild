Welcome!

This site hosts automatically formatted rotation charts of the cards from Love Live SIF All Stars. These charts can be useful for determining at a glance which of the girls is the most likely to receive another card next, based on how many cards are remaining in the current rotation and how long it has been since their last one.

Certainly the developers are not bound to rigid rotations, a fact most evident in event card rotations but frequently seen in other categories of cards as well. So this site should be used as an aid for educated guesses, not a singular source of truth.

Data is retrieved from All Stars Card Database. They do not guarantee full validity of the data at all times as it is automatically harvested from in game announcements, therefore a similar disclaimer applies to this site. Errors may be present but a best effort is made.

Card statistics page uses the original JP release dates for calculating how long it has been since a card was added to the game.

This website is still a work in progress so things are subject to change.

Mobile layout is a pretty new addition so it probably still has things to improve.

Seems to run rather slow on mobile Chrome on Android (but it's fine on Firefox Android), not sure there is much to be done about that with how complex this site's HTML code actually is.

Getting started

Just navigate to any of the categories in the navigation bar on the left.

Note that you can also operate some settings by keybindings:

  • General
  • Q Toggles the idolized switch.
  • W Toggles the reversed order switch.
  • S Cycles to the next source highlight option. Use with the Shift key to go backwards.
  • R Resets above settings back to default.
  • T Toggles the tooltip visibility.
  • Number keys Changes the active sub-page when applicable.
  • Card Stats
  • C Collapses stats table groups. Members are displayed in a single table instead of by group.
  • H Toggles hidden rows for girls that have not yet received a card in the viewed category.

Just navigate to any of the categories in the navigation bar by tapping the bars icon in the top bar.

Some site settings can be modified from the settings menu available by tapping the cog icon.
