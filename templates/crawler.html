SIFAS Card Rotations Crawler Info

This page details information about the sifas-cards crawler.

Crawled pages

This website uses a programmatic crawler to request data from Kirara All Stars Card Database.

Card data is requested via the official API endpoints as documented at here.

Additionally event history data is parsed from the HTML code on both allstars.kirara.ca/jp/history and allstars.kirara.ca/en/history. The number of pages queried is limited to until no new data is found, this generally means only the first page is hit on either address.

Frequency

This crawler automatically visits the aforementioned pages only once per day at 6:05 UTC.

Additional requests may be performed manually in case there are errors present in the local database and cache, and during development of features. Predicting the frequency of requests made in that case is difficult but reasonable efforts are taken to limit number of queries.
